# Compute the Shapley Values and Their Standard Deviation Given `v(S)`

Compute the Shapley Values and Their Standard Deviation Given `v(S)`

## Usage

``` r
compute_estimates(internal, vS_list)
```

## Arguments

- internal:

  List. Not used directly, but passed through from
  [`explain()`](https://norskregnesentral.github.io/shapr/reference/explain.md).

- vS_list:

  List. Output from
  [`compute_vS()`](https://norskregnesentral.github.io/shapr/reference/compute_vS.md).

## Value

The (updated) internal list
