# Function that checks probabilities

Function that checks probabilities

## Usage

``` r
vaeac_check_probabilities(named_list_probabilities)
```

## Arguments

- named_list_probabilities:

  List containing named entries. I.e., `list(a = 0.2, b = 0.9)`.

## Value

The function does not return anything.
