# Gather and Compute the Timing of the Different Parts of the Explain Function

Gather and Compute the Timing of the Different Parts of the Explain
Function

## Usage

``` r
compute_time(internal)
```

## Arguments

- internal:

  List. Not used directly, but passed through from
  [`explain()`](https://norskregnesentral.github.io/shapr/reference/explain.md).

## Value

A list with reformatted timing information.
