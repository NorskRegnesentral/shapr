# Logistic (Cross-Entropy) Loss

Logistic (Cross-Entropy) Loss

## Usage

``` r
log_loss(y, pred)
```

## Arguments

- y:

  Numeric vector of true binary responses (values in 0/1).

- pred:

  Numeric vector (or scalar) of predicted probabilities.

## Value

The mean logistic loss as a single numeric value.
